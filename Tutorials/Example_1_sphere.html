Example 1 Simple Sphere.¶

In this example, we illustrate the use of MPT Calculator for a simple sphere of radius 1, scaled by $\alpha = 0.01$ m, with conductivity $\sigma_* = 10^6$ S/m, and $\mu_r = 1$ over the frequency range $10^1 \ge \omega \ge 10^8$ rad/s using the full order solver and a single processor using an order 3 approximation and a coarse discretisation.

Defining material properties¶

The object in question is defined in OCC_sphere.py following the format defined in Tutorial 5. To update the conductivity or permeability, change lines 13 and 14 to the appropriate values.

Recall that the OCC geometries must be saved in the $\texttt{/OCC_Geometry/}$ folder and that the mesh saved on line 44 must match the file name of the python file. In this example that is $\texttt{VolFiles/OCC_sphere.vol}$.

Defining object size and frequency sweep¶

Here, we specify that we want to consider a sphere with $\alpha=0.01$ m and the frequency range $10^1 \ge \omega \ge 10^8$. In this example, we wish to consider 20 logarithmically spaced frequencies.

geometry = 'OCC_sphere.py'
alpha = 0.01
start_stop = (1,8,20)
order = 3
mesh_coarseness = 'coarse'
